AA Property Services

Proyecto intermodular de DAW basado en el desarrollo de una solución digital para una empresa de servicios premium orientada a villas y propiedades.

 Descripción

AA Property Services es un proyecto académico con base en un caso real. Su objetivo es desarrollar una solución escalable que comience con una web corporativa y pueda evolucionar hacia una plataforma de gestión de servicios, incidencias, mantenimiento y comunicación con propietarios.

Estado actual

En esta fase del proyecto se ha desarrollado la parte web corporativa, incluyendo:

- página de inicio
- página de servicios
- página sobre nosotros
- página de contacto

También se ha comenzado la organización de la documentación del proyecto y del material recibido de la clienta.

Tecnologías utilizadas

- HTML5
- CSS3
- Git
- GitHub

Estructura del repositorio

- `web/` → páginas HTML, estilos CSS e imágenes de la web
- `docs/` → documentación del proyecto y material de análisis
- `docs/cliente/` → formularios, logos y documentación aportada por la clienta

Objetivo del proyecto

Desarrollar una base profesional y bien organizada para un proyecto intermodular de DAW, integrando progresivamente la parte visual, la documentación, la base de datos y la futura lógica de aplicación.
