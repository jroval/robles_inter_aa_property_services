Proceso de reserva

Cómo se reservan los servicios

  1. El cliente selecciona el servicio que desea contratar.
  2. Elige una fecha, hora o franja disponible.
  3. La solicitud queda pendiente de aprobación.
  4. Recibe confirmación una vez validada por la empresa.

Las cancelaciones deben realizarse con la antelación mínima indicada para cada servicio.
